# Mini-Quiz-Application---Android
1. Problem Definition:
In today's digital learning environment, students require interactive tools to evaluate their academic knowledge. Traditional learning methods such as reading textbooks do not always provide immediate feedback regarding a student's understanding of a subject.
The problem addressed in this project is the lack of a simple and portable platform where students can quickly test their knowledge using multiple-choice questions.
To solve this problem, the Mini Quiz App is developed as a simple Android application that simulates a quiz system. The application allows users to answer multiple-choice questions and instantly view their scores. This helps students practice questions, evaluate their performance, and improve their understanding in an interactive way.

2. Objective of the Application:
The main objectives of the Mini Quiz Application are:
• Develop a simple Android application with multiple screens
• Allow users to answer quiz questions using RadioButtons
• Display quiz questions in an easy-to-understand interface
• Calculate the final score automatically
• Display the result screen after quiz completion
• Demonstrate Android concepts such as activities and explicit intents
• Create a simple and user-friendly learning tool for students

3. Key Features of the Application:
The application provides the following features:
Welcome screen displaying the application title
Get Started button to begin the Quiz app
Select the Subject from the given subjects
Quiz screen showing questions and multiple-choice answers
Radio buttons for selecting answers
Automatic score calculation
Result screen displaying the final score
Option to restart the quiz
Smooth navigation between screens
Simple and clean user interface

5. Application Flow:
The application follows the workflow below:
The application starts with the Welcome Screen
user clicks the Get Started button.
Subject Selection screen appears
Select any Subject
The Quiz Screen appears displaying the questions.
The user selects an answer using RadioButtons.
After completing all questions, the user clicks Submit Button.
The application calculates the score.
The Result Screen displays the final score.
The user can Restart the quiz or Exit the application.

6. Conclusion:
The Mini Quiz Application demonstrates the basic functionality of an interactive quiz system. The application allows users to answer multiple-choice questions and instantly view their scores.
Through this project, we learned how to develop Android applications using activities, explicit intents, UI components, and Android resources.
